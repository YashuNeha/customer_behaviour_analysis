# customer_behaviour_analysis

# 📊 Data Analytics Project – End-to-End Workflow

## 🔍 Overview

This project demonstrates a complete data analytics workflow, starting from raw data ingestion to delivering actionable insights through visualization and reporting. It covers data cleaning, exploratory data analysis (EDA), SQL-based querying, and dashboard creation.

The goal of this project is to showcase practical data analytics skills used in real-world business scenarios.

---

## 📁 Dataset

* The dataset contains customer transaction and shopping behavior data.
* It includes features such as:

  * Customer details (ID, gender, age)
  * Product/category information
  * Purchase quantity and pricing
  * Transaction dates and ratings

---

## 🛠️ Tools & Technologies

* **Python** (Pandas, NumPy, Matplotlib/Seaborn) – Data loading, cleaning, and EDA
* **MySQL** – Data storage and query analysis
* **Power BI** – Interactive dashboard creation
* **Jupyter Notebook** – Development and analysis environment

---

## ⚙️ Project Steps

### 1. Data Loading

* Imported the dataset using Python (Pandas)
* Handled encoding and file format issues

### 2. Data Cleaning

* Removed null and duplicate values
* Corrected data types (dates, numeric fields)
* Standardized column names and formats

### 3. Exploratory Data Analysis (EDA)

* Analyzed customer demographics
* Identified top-performing products and categories
* Explored sales trends over time
* Visualized key patterns using charts and plots

### 4. SQL Analysis (MySQL)

* Imported cleaned data into MySQL
* Wrote queries to:

  * Find top-selling products
  * Calculate average ratings
  * Analyze category-wise performance
  * Identify customer purchase patterns

### 5. Dashboard Creation (Power BI)

* Built an interactive dashboard with:

  * Sales overview
  * Category performance
  * Customer insights
  * Trend analysis

### 6. Reporting

* Summarized key insights from analysis
* Highlighted business recommendations
* Presented findings in a structured format

---

## ▶️ How to Run the Project

1. **Clone the repository**

   ```bash
   git clone <your-repo-link>
   cd <project-folder>
   ```

2. **Run Python scripts / Jupyter Notebook**

   * Load dataset
   * Perform cleaning and EDA

3. **Set up MySQL**

   * Create database and table
   * Import cleaned dataset
   * Run SQL queries

4. **Open Power BI**

   * Load dataset or connect to MySQL
   * Open the `.pbix` dashboard file
